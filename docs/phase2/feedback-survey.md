# Post-Workshop Feedback Survey

> Collect participant feedback immediately after the workshop to improve future cohorts.

---

## Implementation Notes

**Recommended platform:** Google Forms (free) or Typeform (free tier, nicer UX)

**When to send:** Include the form link / QR code in the closing ceremony. Post the link in the workshop Discord channel the same night.

**Target response rate:** 80%+ while participants are still in the room or within 24 hours.

**Setting up in Google Forms:**
1. Go to forms.google.com → Blank form
2. Title: "Vibe Coding Workshop — Your Feedback"
3. Add sections (one per section below) using the "Add section" button
4. For rating scales, use "Linear scale" question type (1–5)
5. For multiple choice, use "Multiple choice" question type
6. Set all questions to "Required" except testimonial and open-ended

---

## Survey Structure

**Estimated completion time:** 5–7 minutes

**Total questions:** 15 + 2 optional sections

## Success Metrics to Track

Track these numbers after each cohort to improve over time:

| Metric | Target | How to Measure |
|--------|--------|---------------|
| Response rate | 80%+ | (responses / participants) × 100 |
| Overall rating average | 4.3+ out of 5 | Average of Q1 |
| Net Promoter Score proxy | 80%+ "Yes, recommend" | Q2 responses |
| "Shipped all 4" rate | 70%+ | Q7 responses |
| Confidence after workshop | 85%+ positive | Q15 responses |
| Testimonial consent | 60%+ | Optional Section A |

## Sample Survey Link Setup (Google Forms)

```
1. Create form at forms.google.com
2. Add all 15 questions as described above
3. Go to Responses tab → Enable "Collect email addresses" (optional)
4. Click Send → Get link → Shorten URL
5. Create a QR code at qr-code-generator.com
6. Add QR code to closing ceremony slide
7. Post raw link in workshop Discord channel
```
