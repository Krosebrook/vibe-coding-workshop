# Phase 2 — Workshop Operations & Participant Experience

> Complete operational documentation for delivering the Vibe Coding Workshop and supporting participants before, during, and after the day.

---

## What's in Phase 2

Phase 2 covers everything needed to run the workshop day and keep participants engaged after they leave. It includes the instructor guide, participant materials, tool setup guides, community infrastructure, and all communications.

## How Phase 2 Fits the Workshop Lifecycle

```
Before the Workshop
  └── email-sequences/pre-workshop-onboarding.md
  └── setup-guides/ (participants complete on their own)
  └── github-classroom-setup.md (instructor sets up)

Day of Workshop
  └── workshop-day-schedule.md (instructor's timeline)
  └── instructor-guide.md (instructor's companion)
  └── participant-workbook.md (printed or digital for each participant)
  └── prompting-playbook.md (reference card at each seat)

After the Workshop
  └── feedback-survey.md (sent same night)
  └── email-sequences/post-workshop-followup.md
  └── community/discord-guide.md
  └── community/alumni-newsletter-template.md
```
