# Setup Guides — Overview

> Step-by-step tool setup guides for workshop participants. Complete these before workshop day.

---

## Why These Exist

The workshop moves fast. Participants who arrive already set up can spend the day building. Participants who spend the morning on signups miss the best parts.

These guides are written for complete beginners — no assumed technical knowledge.

---

## Tools Overview

| Tool | Purpose | Difficulty | Est. Setup Time | Guide |
|------|---------|------------|-----------------|-------|
| **Cursor** | AI-powered code editor — where you'll spend most of the day | Easy | 10–15 min | [cursor-setup.md](./cursor-setup.md) |
| **Claude** | The AI that writes and explains your code | Easy | 5 min | [claude-setup.md](./claude-setup.md) |
| **v0** | Generates UI components from text descriptions | Easy | 5 min | [v0-setup.md](./v0-setup.md) |
| **Vercel** | Deploys your app to a live URL | Easy | 5 min | [vercel-setup.md](./vercel-setup.md) |
| **Supabase** | Database for your app (used in advanced projects) | Medium | 15–20 min | [supabase-setup.md](./supabase-setup.md) |
| **GitHub** | Stores your code; required for Vercel | Easy | 5 min | — (covered in Cursor setup) |

**Total estimated setup time:** 45–60 minutes

---

## Setup Order

Complete these in order — some tools depend on others.

```
1. GitHub (create account at github.com)
   ↓
2. Cursor (download + install + connect to GitHub)
   ↓
3. Claude (create account)
   ↓
4. v0 (sign up with GitHub)
   ↓
5. Vercel (sign up with GitHub)
   ↓
6. Supabase (create account — optional before the workshop)
```

---

## Minimum Required Before Workshop Day

You must have these working before arriving:
- [ ] GitHub account created
- [ ] Cursor installed and open on your laptop
- [ ] Claude account created and you can get a response
- [ ] Vercel connected to your GitHub account

Supabase and v0 can be set up during the workshop if needed — they're quick.

---

## Getting Help

If you get stuck during setup, email `hello@vibecodingworkshop.com` before the workshop. Common issues are solved in under 5 minutes with the right information.
